# 🐦 Neon Flappy

משחק Flappy Bird בעיצוב נאון — קובץ HTML יחיד, בלי תלויות, עובד גם אופליין וגם בנייד.

## משחק
פתח את `index.html` בדפדפן, או שחק אונליין דרך GitHub Pages.

**שליטה:** לחיצה על המסך / רווח / חץ למעלה — כדי לעוף.

## תכונות
- ציפור נאון זוהרת עם אנימציה
- צינורות עם אפקט זוהר
- רקע כוכבים נע (אפקט עומק)
- שמירת שיא מקומית (localStorage)
- תמיכה במגע ובמקלדת
